# VirusHack-Uma.Tech

В процессе решения задачи было создано расширение `Pass Media Assistent` для браузера Google Chrome

![Extension](https://sun6-16.userapi.com/Yfglvs9I1Q1_kIY4b9OzMcxmwUKZVdKzGPVH_A/PxZLgwQMtaY.jpg)

## Features

* Расширение упрощяет процедуру авторизации на сайте [pass.media](https://pass.media/) для людей с ограниченными возможностями посредством добавления голосового помощника
* Расширение позволяет изменять характеристики любого видео на сайте (т.е. не нужна предобработка), такие как яркость, контрастность, насыщенность, и т.д.
А так же позволяет использовать различные SVG фильтры

### Usage

В репозитории хранится не запакованное расширение, которое можно добавить в браузер Google Chrome, для этого нужно:
1. Скачать расширение
2. Открыть страницу управления расширениями [chrome://extensions](chrome://extensions)
3. Включить режим разработчика
4. Нажать кнопку **LOAD UNPACKED** и выбрать папку с расширением
  ![Loading](https://developer.chrome.com/static/images/get_started/load_extension.png)

### Голосовой помощник
Для голосового взаимодействия пользователя с сайтом было выделено три уровня:
1. Speech recognition layer - конвертация речи пользователя в текст
2. Natural language processing layer - парсинг валидной команды (Ввод логина,  пароля, и т.д.)
3. UI Interaction - взаимодейтсвие с интерфейсом (подстановка логина, пароля)
  Так как действия пользователя должны быть последовательными, то этот слой содержит в себе конечный автомат, который хранит в себе текущее состояние и возможные переходы.
